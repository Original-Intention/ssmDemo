# 简介
本人自身用于学习SSM框架与实践的一个小DEMO,希望这个小DEMO能对大家学习有帮助

# 使用技术 

IoC容器:spring

web框架:springmvc

orm框架:mybatis

数据源:dbcp2

日志: sl4j

Json: Gson

# 起步:

1.初始化项目

1)下载Mysql,创建一个数据库名字为giit,导入create.sql与populate.sql,直接运行其中sql即可

2)下载Tomcat

3)使用Intellij Idea导入项目,之后配置项目启动方式,使用刚才下载好的Tomcat
  
4)运行
